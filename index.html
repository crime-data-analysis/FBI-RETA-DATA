FBI Return A Data Recode

Python scripts for recoding FBI Return A Master Files for data analysis.

View on GitHub Download .zip Download .tar.gz

Welcome to Crime Data Analysis! This repo is specifically geared towards recoding and analyzing FBI Return A Master File dat. It is really just a collection of the Master Files data & a few Python 3 scripts that recode them and load them into Jupyter for analysis.

It's not clear why the files have occassional non-ascii characters; for now the scripts replace them with ascii.
